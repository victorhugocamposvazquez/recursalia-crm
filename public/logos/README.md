# Logos para el PDF del curso

Coloca aquí las imágenes de los logos para la **portada** y **contraportada** del PDF del curso (ebook):

- **recursalia.png** o **recursalia.jpg** — Logo de Recursalia
- **hotmart.png** o **hotmart.jpg** — Logo de Hotmart

Si no hay archivos en esta carpeta, el PDF usará el nombre de la marca en texto.

Recomendado: PNG con fondo transparente, tamaño aproximado 400×120 px o similar (se escalan automáticamente).
